## تبدیل متن به نمودار BPMN

یک برنامه‌ی سبک مبتنی بر Flask که توضیحات متنی فرایند را به نمودار خطی BPMN تبدیل کرده و در مرورگر نمایش می‌دهد.

### پیش‌نیازها

- Python 3.10 یا جدیدتر
- نصب `pip`

### نصب و اجرا

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python run.py
```

سپس در مرورگر به آدرس <http://127.0.0.1:5000> مراجعه کنید.

### نحوه استفاده

1. شرح فرایند را به صورت مرحله به مرحله وارد کنید (هر مرحله را با نقطه یا خط جدید جدا کنید).
2. روی «تبدیل به نمودار» کلیک کنید.
3. نمودار BPMN در سمت راست صفحه نمایش داده می‌شود.

> **نکته:** این نسخه دمویی است و مراحل را به صورت خطی به BPMN تبدیل می‌کند. برای تشخیص ساختارهای پیچیده (انشعاب، تصمیم‌گیری و ...) باید الگوریتم NLP پیشرفته‌تری توسعه یابد.
